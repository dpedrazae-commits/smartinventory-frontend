#  Smart Inventory — Frontend (Angular)

Interfaz moderna construida con Angular para la plataforma Smart Inventory.

##  Funcionalidades
- Login y autenticación
- Dashboard con KPIs
- Listado de productos
- Movimientos de inventario
- Gráficas dinámicas
- Escaneo QR
- Insights generados por IA

## Tecnologías
- Angular 16+
- TypeScript
- SCSS
- ng2-charts / ApexCharts
- Angular Material

## Ejecución
```bash
npm install
ng serve
